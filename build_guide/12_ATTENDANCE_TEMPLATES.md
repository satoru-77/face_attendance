# 12 — Attendance Templates

> **Goal:** Build attendance record views, daily/monthly reports, and the student self-attendance calendar.
> **Files to create:**
> - `templates/attendance/records.html`
> - `templates/attendance/daily_report.html`
> - `templates/attendance/monthly_report.html`
> - `templates/attendance/my_attendance.html`

---

## File 1 — `templates/attendance/records.html`
Filterable table of daily attendance records with status badges and department filters.

## File 2 — `templates/attendance/daily_report.html`
Day-level summary with hourly chart (Chart.js), present/absent/late counts.

## File 3 — `templates/attendance/monthly_report.html`
Per-student monthly breakdown with percentage bars and CSV export button.

## File 4 — `templates/attendance/my_attendance.html`
Student's personal attendance calendar grid with color-coded days.
